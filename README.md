# Jarvis — Assistente Android

Uma tentativa de criar um assistente pessoal pra Android inspirado no Jarvis do Tony Stark. Integrei IA generativa via API da Groq pra responder perguntas e fazer automações básicas.

Esse foi um dos projetos mais desafiadores que já encarei — foi minha primeira experiência com Kotlin e desenvolvimento Android. Errei bastante, principalmente com permissões e ciclo de vida das activities, mas aprendi muito no processo.

Ainda tem muita coisa que eu melhoraria, mas foi um projeto que me abriu os olhos pra como apps móveis funcionam por dentro.

## O que aprendi
- Desenvolvimento Android com Kotlin
- Ciclo de vida de Activities e Fragments
- Integração com APIs externas (Groq)
- Como lidar com permissões no Android

## Requisitos
- Android Studio Hedgehog ou superior
- Java 17+
- Android SDK 34
- Dispositivo Android 8.0+ (API 26)
- Conta Groq (gratuita)

## Stack
Kotlin · Android SDK · Groq API
